# OTTO Lab

Observational Target Trial Operations (OTTO), School of Psychological Sciences, Te Herenga Waka — Victoria University of Wellington.

OTTO Lab develops and applies observational causal inference — target trial emulation and outcome-wide analysis — to questions in psychological science.

## Site

- Quarto source lives in this repository.
- Published at <https://go-bayes.github.io/ottolab/> (the canonical domain <https://ottolab.org> is pending DNS configuration).

## Build

```sh
quarto render        # renders the site to docs/
quarto preview       # local preview with live reload
```

GitHub Pages serves the site from the `main` branch `/docs` folder.
